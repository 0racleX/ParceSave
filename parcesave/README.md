# ParceSave

ParceSave is a digital application that connects urban and agricultural food surplus with consumers, NGOs, and food banks, to transform food waste into economic, social, and environmental opportunities. This project was built for a hackathon using Expo and React Native.

**Status:** Prototype / Hackathon demo

## 🚨 The Problem

- **Global:** 783 million people suffer from hunger, while nearly 1/5 of the food produced is wasted.
- **Colombia:** 9.7 million tons of food are lost annually (equivalent to 34% of the national food supply).
- **Medellín:** Around 865 tons of food are discarded daily—enough to feed 70,000 families (approx. 900,000 meals) for a day.

## 💡 The Solution (How it Works)

1. **Surplus Registration**: Farmers and local businesses can easily register their excess food.
2. **Geolocation & Visibility**: Connects nearby surplus with potential buyers or donees.
3. **Action Options**: Users can choose to sell food at a discount or donate it.
4. **Hybrid Logistics**: Integrates pickup, urban micro-deliveries, and transport from the countryside.
5. **Predictive AI**: Suggests the best time to sell, donate, or redistribute, optimizing routes and inventory.
6. **Impact Reports**: Provides real-time metrics on saved food, benefited families, and avoided CO₂.

## 🌟 Value Proposition

- **Integral Urban-Agricultural Model:** Rescues food both in the city (supermarkets/restaurants) and in the field (harvest/post-harvest).
- **Hybrid Sale & Donation:** A single platform handling discounted purchases and traceable donations for CSR (Corporate Social Responsibility) and tax benefits.
- **Measurable Impact:** Real-time reporting on the social and environmental impact generated.
- **Strategic Alliances:** Connections with local governments and initiatives like "Hambre Cero".

## 🎯 Target Market

1. **Final Consumers:** Households facing food insecurity or seeking affordable products.
2. **Urban Commerce:** Supermarkets, minimarkets, and restaurants.
3. **Farmers & Rural Associations:** Small and medium producers, cooperatives.
4. **Social Organizations & Governments:** NGOs, city halls, and public sustainability programs.

## 💼 Business Model

- **Subscription Plans** for commerce and farmers to publish unlimited surplus.
- **Premium Impact Services** providing custom ESG/CSR reports for companies and NGOs.
- **Responsible Advertising** for sustainable brands within the app.
- **Public Contracts & Alliances** to manage surplus and donations in partnership with local governments.
- **Free access** for final consumers.

## 🛠️ Technical Stack & Quick Start

### Highlights

- Fast prototype using Expo and React Native
- Navigation with `@react-navigation/native` and `native-stack`
- HTTP requests via `axios`
- Screens: Login, Explore, Product Detail, Cart, Dashboard

### Quick Start

1. Open a terminal and change to the project folder:

```bash
cd parcesave
```

2. Install dependencies:

```bash
npm install
# or `yarn`
```

3. Start the dev server (Expo):

```bash
npx expo start
# or `npm run start`
```

4. Run on a device/emulator:

```bash
npm run android
# or `npm run ios` / `npm run web`
```

## 📁 Project Structure

- `App.js` — root component and navigation setup
- `index.js` — Expo entry
- `assets/` — images and static files
- `screens/` — feature screens (Login, Explore, ProductDetail, Cart, Dashboard)
